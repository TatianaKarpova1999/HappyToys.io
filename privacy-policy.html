Política de Privacidade - ArsCommodum

Coleta de Informações Pessoais

A ArsCommodum coleta apenas as informações essenciais para processar pedidos, oferecer nossos serviços e aprimorar a experiência de nossos clientes, sempre respeitando a privacidade e confidencialidade de todos os dados fornecidos.

Uso dos Dados

As informações pessoais coletadas serão usadas exclusivamente para facilitar transações, realizar entregas de produtos e manter comunicação com relação a atualizações ou melhorias nos nossos serviços.

Armazenamento de Dados

As informações pessoais são armazenadas de forma segura e protegidas contra acessos não autorizados ou perdas acidentais, em conformidade com os padrões avançados de proteção de dados.

Compartilhamento de Dados com Terceiros

A ArsCommodum não compartilha, vende ou distribui as informações pessoais dos clientes para terceiros, salvo quando exigido por lei ou para a execução de serviços essenciais.

Consentimento Explícito

O processamento de dados pessoais é baseado no consentimento explícito dos clientes, garantindo clareza e conformidade com as regulamentações de proteção de dados.

Direitos dos Titulares dos Dados

Os clientes têm o direito de acessar, corrigir ou solicitar a exclusão de seus dados pessoais a qualquer momento, entrando em contato formalmente com a ArsCommodum.

Medidas de Segurança

A ArsCommodum adota medidas rigorosas, tanto técnicas quanto administrativas, para proteger as informações pessoais dentro de nossos sistemas.

Política de Retenção de Dados

Os dados pessoais serão retidos apenas pelo tempo necessário para cumprir as finalidades para as quais foram coletados, ou conforme exigido por lei.

Atualizações da Política

Esta política de privacidade pode ser revisada periodicamente, e qualquer alteração será comunicada aos clientes por meio dos canais apropriados.

Responsabilidade pelo Tratamento de Dados

A ArsCommodum se compromete a gerenciar os dados pessoais de forma responsável, em conformidade com as leis vigentes e princípios éticos de confidencialidade.

Transferências Internacionais de Dados

Os dados pessoais não serão transferidos para fora das fronteiras nacionais sem o consentimento explícito do cliente, salvo quando exigido por lei.

Finalidade do Tratamento de Dados

Os dados pessoais são processados exclusivamente para execução de contratos, entrega de produtos e para melhorar a experiência do cliente com os serviços da ArsCommodum.

Proteção de Menores

A ArsCommodum não coleta dados conscientemente de menores de idade sem o consentimento dos responsáveis legais, cumprindo totalmente as leis de proteção de dados infantis e juvenis.

Comunicação Direta

Para dúvidas, solicitações ou preocupações relativas ao uso dos dados pessoais, os clientes podem entrar em contato diretamente com a ArsCommodum pelo e-mail arscommodum@gmail.com.

Legislação Aplicável

Esta política de privacidade é regida pelas disposições das leis de proteção de dados aplicáveis, incluindo a Lei Geral de Proteção de Dados Pessoais (LGPD).
